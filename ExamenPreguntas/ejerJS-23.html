Ejercicio No 23


You need to test the value of the following variable in JavaScript.

var length = "75";

A block of code must execute if the length equals 75 regardless of the data type. You need to use the statement that meets this requirement.

Which lines of code should you use? (Each correct answer presents a complete solution. Choose two.)

  • A. if (length = = = 75)
  • B. if (length = = 75)
  • C. if (length! = 75)
  • D. if (length = = "75")


Correct Answer: BD

Section: Volume A

Explanation

Explanation/Reference:

Explanation:

When comparison is made using double-equals operator (==), it will check the values of variable and convert them to a common type and returns true if both are equals. So comparing number with string having the same value will return true.

Examples:

1
console.log(23 == "23"); // true 2
console.log(1 == true); // true

Explicación

Cuando la comparación se realiza usando el operador de doble igual (==), verificará los valores de la variable y los convertirá a un tipo común y devolverá verdadero si ambos son iguales. Entonces, comparar el número con una cadena que tenga el mismo valor devolverá verdadero

Por discriminación la opcion C no es correcta ya que la condición dice que hay que ejecutar el código si la longitud es igual a 75, aquí no es el caso ya que es desigual.

La primera opción o "===" es una comparación en valor y tipo por lo que tampoco es correcta. La demostración se hace en esta página por medio de alert donde se prueban cada una de las opciones para confirmar que las respuestas B y D son correctas.

Quedan las B y la C. La opción B es correcta ya que compara solo en valor. La opción D es correcta ya que compara en tipo, que en este caso es cadena.

Respuesta:

Las opciones B y D.

Fuentes:

Las mismas expuestas en la respuesta dada.
