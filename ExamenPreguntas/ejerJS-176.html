Explicación

En el modelo de registro de eventos en línea, los controladores de eventos se agregan como atributos a los elementos HTML en los que estaban trabajando, como:

<A HREF="somewhere.html" onClick="alert('¡He hecho clic !')">

Cuando se produce un evento de clic en el enlace, se invoca el controlador de eventos y se ejecuta el script: se muestra una alerta. También puede invocar una función de JavaScript:

<A HREF="somewhere.html" onClick="doSomething()">

El método event.stopPropagation () detiene la propagación de un evento a los elementos principales, lo que evita que se ejecuten los controladores de eventos principales.

Es útil usar el método event.isPropagationStopped() para comprobar si este método fue llamado para el evento.
