Ejercicio No 261

You want to create an expando attribute on several <h3> tags that display vehicles for sale. The expando attribute will store the VIN (vehicle identification number) of the vehicle for sale.
Which of the following is the most appropriate example of creating the expando attribute?

  • A. <h3 vin='current VIN here'>1965 VW Beetle</h3>
  • B. <h3 id='current VIN here'>1965 VW Beetle</h3>
  • C. <h3 data-vin='current VIN here'>1965 VW Beetle</h3>
  • D. <h3 datavin='current VIN here'>1965 VW Beetle</h3>


Correct Answer: C
Section: Volume D Explanation

Explanation/Reference:
References: http://www.daoudisamir.com/references/vs_ebooks/html5_css3.pdf




Explicación

expando object: Establece o devuelve un valor booleano que indica si se pueden utilizar propiedades adicionales dentro del objeto. Una propiedad expando es una propiedad personalizada. De forma predeterminada, todos los navegadores comunes admiten la adición de propiedades arbitrarias a cualquier objeto, pero solo Internet Explorer permite desactivarlo.

  • Con la propiedad expando del documento, puede establecer o recuperar si el soporte de propiedad personalizada está habilitado.
  • Con la propiedad expando de un atributo, puede recuperar si el atributo es una propiedad personalizada o no.
Según el documento MDN :

HTML5 está diseñado con la extensibilidad en mente para los datos que deben asociarse con un elemento en particular, pero no necesitan tener un significado definido. Los atributos data- * nos permiten almacenar información adicional en elementos HTML semánticos estándar sin otros trucos como atributos no estándar, propiedades adicionales en DOM o Node.setUserData ().
